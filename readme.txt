== Description ==

This plugin use for wordpress comments validation to the comment form. easy to install and only need to activate the plugin.

= Features =
1. Easy to install.
2. No need to settings in admin panel.

= Requirements =
This plugin requires PHP 5 or PHP 7 and WordPress 3.0 or greater. It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

= Contact =
* If you have any suggestion, feel free to email me at upen.itm@gmail.com.

== Installation ==

1. Go to Plugins > Add New. 
2. Searh plugin keyword "wp comment form validation".
3. install and activate the WordPress Plugin.
